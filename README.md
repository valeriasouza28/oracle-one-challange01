# Decodificador de Texto

## Descrição

A aplicação cripitografa um tesxto, e descripitografa um texto inserido pelo usuário, 
e tem a opção de copiar esse texto para área de tranferência para que possa ser colado 
em outro lugar.

- CSS
- Javascript
